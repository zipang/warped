warped
======

a jQuery plugin for silly *'warped'* text effects.
Made at the Open World Forum 2012 to kill time waiting for the technical Openshift presentation.

Usage example:
```js
// Warp the level 1 and level 2 titles
$("h1, h2").warped();
```
